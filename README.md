A minimal, light-weight **Rule-Based Chatbot** built in Python that responds to predefined inputs using a structured hashmap configuration for optimal efficiency and safe fallbacks.

---

### ✨ Features

* **⚡ Optimized Routing**: Uses an O(1) hashmap lookup rather than verbose `if-else` branches.
* **🛡️ Safe Fallback**: Built-in default response engine when inputs are not recognized.
* **🔁 Interactive Console Loop**: Continuous user chat loop with instant responses.

---

### 🚀 Quick Start

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Sid2403-max/Chatbot-Predefined-inputs-Decode-lab-.git
   cd Chatbot-Predefined-inputs-Decode-lab-
   ```

2. **Run the chatbot**:
   ```bash
   python chatbot.py
   ```

3. **To Exit**:
   Type `exit` in the terminal to end the session.
